## Prerequisites
- npm 
```npm install npm@latest -g```
- mongoose
- nodemon
- express
- dotenv
- bcrypt
- validator

## Installation
Clone repo :
- ```git clone git@github.com:krabandicoot/Planty_Becode.git ```
Install npm package : 
- ```cd ./app/client && npm install```
- ```cd ./app/server && npm install```
